Gearbox Sizing — Load Spectrum Method

Optional calculation — requires nominal crane torque M_Nenn. Determines the required gearbox nominal torque rating using the actual load spectrum rather than a fixed safety factor.

GB1Gearbox torque — nominal

{% latexify fg1 math_block=True %}

Same as drivetrain Step 2.

GB2Gearbox torque — maximum

Same as drivetrain Step 1.

GB3Load spectrum ratio

{% latexify fg3 math_block=True %}
λ dimensionless ratio between nominal and maximum gearbox torque.
λ = 1 means pure peak load; λ → 0 means rarely loaded.

GB4Required gearbox nominal torque

{% latexify fg4 math_block=True %}
{% latexify fg4_exp math_block=True %}
M_gear,soll = required catalogue nominal torque rating [Nm]
Select a gearbox with rated torque ≥ M_gear,soll.

Compared to the simple safety-factor method (M2_nom × 1.34), this approach accounts for the actual duty cycle, giving a more accurate — sometimes higher, sometimes lower — required rating.
