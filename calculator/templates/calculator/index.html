Formula Reference (click to expand/collapse)

Complete drivetrain sizing chain: Motor → Bevel Gearbox → Worm Gear → Slewing Ring. Based on the Palfinger / Z-Systems sizing sheets.

{# Step 1 #}

1Worm shaft torque — maximum

# M_Max = max crane slewing torque (Nm) # i_worm = worm gear ratio # η = worm gear efficiency (0–1) M2_Max = M_Max / (i_worm × η)

Reflects the full peak torque the worm shaft must transmit. Worm efficiency typically 0.35–0.50 for single-start worms.

{# Step 2 #}

2Worm shaft torque — nominal

# M_Nenn = nominal (continuous) crane torque M2_Nenn = M_Nenn / (i_worm × η)

Used for thermal / continuous-duty sizing of the gearbox.

{# Step 3 #}

3Required gearbox torque

# S_f = safety factor = 1.34 (Z-Systems) M_gear_req = M2_Nenn × 1.34

The bevel gearbox catalogue torque rating must be ≥ M_gear_req. Example: ZK4 rated 760 Nm > 517 Nm required.

{# Step 4 #}

4Bevel gearbox ratio

# n_motor = motor speed (rpm) # n_gear_out = bevel output shaft speed (rpm) i_bevel = n_motor / n_gear_out

The bevel gearbox reduces motor speed to the worm input shaft speed.

{# Step 5 #}

5Slewing speed

# n_gear_out = bevel output = worm input speed n_slew = n_gear_out / i_worm

Final crane rotation speed in rpm. Typical target: 0.25–0.35 rpm for offshore cranes.

{# Step 6 #}

6Motor torque required

# Back-calculated from worm max input torque M_motor_req = M2_Max / i_bevel

The motor must be able to produce this starting torque to move the crane under peak load.

{# Step 7 #}

7Motor starting torque available

# M_n = motor rated (nameplate) torque # Ma/Mn = locked-rotor / rated torque ratio M_start = M_n × (Ma/Mn)

For IEC motors Ma/Mn is typically 2.0–3.5 for standard induction motors and up to 4+ for high-torque designs.

{# Step 8 #}

8Torque feasibility check

margin = M_start / M_motor_req # margin ≥ 1.3 → OK # margin ≥ 1.1 → Marginal (not recommended) # margin < 1.1 → Too small → choose bigger motor

The 1.3× margin accounts for supply-voltage dip, temperature, bearing stiction, and manufacturing tolerances.

{# Step 9 #}

9Motor power required

# 9550 = 60 × 1000 / (2π) — unit conversion constant # P in kW, M in Nm, n in rpm P_motor = (M_motor_req × n_motor) / 9550

Minimum nameplate kW rating of the selected motor. Always round up to the next standard IEC frame size.

{# end formula row #}
Supplier Data Check Criteria
  • Torque / power values: supplier value ÷ required ≥ 1.10 → OK, ≥ 1.00 → Marginal, < 1.00 → Too small
  • Ratio values: |supplier − calculated| ÷ calculated ≤ 2% → OK, otherwise Mismatch
  • Motor starting torque checked against M_motor_req (Step 6)
  • Gearbox torque checked against M_gear_req (Step 3)
  • Motor power checked against P_motor (Step 9)
